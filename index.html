调用API

青悦空气质量API采用阿里云API Gateway服务搭建，API调用方式详见http://docs-aliyun.cn-hangzhou.oss.aliyun-inc.com/pdf/apigateway-c_user-guide-cn-zh-2016-08-25.pdf

示例sdk

https://github.com/aliyun/api-gateway-demo-sign-python

示例代码

from com.aliyun.api.gateway.sdk import client
from com.aliyun.api.gateway.sdk.http import request
from com.aliyun.api.gateway.sdk.common import constant

host = "http://api.epmap.org"
url = "/api/v1/air?longitude=121.260372&latitude=31.09805"

req = request.Request(host=host, url=url, method="GET", time_out=30000)
req.set_content_type(constant.CONTENT_TYPE_JSON)

#cli = client.DefaultClient(app_key="23436472", app_secret="049a263c4fcc7f9cad844d1072e55e57")

cli = client.DefaultClient(app_key="keysecret2", app_secret="secretsecret2")

print cli.execute(req)
